# Chat simple

## Introduccion

> Aplicacion de chat creado con Angular y Express para aprender el uso de SOCKET IO, deploy en Heroku y Vercel

## Tecnologias

> Angular Material, Socket io, Express y Typescript

## Dato

### Solamente poner nombre y empezá a escribir
> La aplicacion hiberna en Heroku, por lo tanto si no anda, tarda 20 segundos en conectar.
